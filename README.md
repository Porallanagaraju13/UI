## ⚙️ Getting Started

### 1. Clone the Repository
```bash
git clone https://github.com/Porallanagaraju13/UI.git
cd UI
```

### 2. Open Any Project
Simply open the `index.html` of any project in your browser:
```bash
# On Windows
start project-1/index.html

# On Mac/Linux
open project-1/index.html
```

No build tools or dependencies required — pure HTML, CSS, and JavaScript.
